# OCI Vault

Terraform root for AlMadar secrets in OCI Vault.

## Stored Secrets

For each environment (`dev`, `test`, `prod`) this stack stores JSON secrets for:

- PostgreSQL credentials
- JWT secrets
- Strapi secrets
- S3-compatible Object Storage credentials

Secret values are supplied through `terraform.tfvars`, which is ignored by Git, or generated by Terraform using `random_password`.

## Important

Terraform state will contain secret material because OCI Vault secret payloads are managed by Terraform. Store state only in the approved encrypted remote backend.

## Usage

```bash
cd infrastructure/terraform/environments/vault
cp terraform.tfvars.example terraform.tfvars
terraform init
terraform apply
```

After apply, use `terraform output secret_names` and `terraform output vault_id` to configure External Secrets manifests.
